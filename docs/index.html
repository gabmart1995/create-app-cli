Create App Cli

Un CLI con Esteroides

Linea de comandos

Una linea de comandos diseñado para el desarollador web, que le permite iniciar rapidamente con plantillas de los proyectos mas demandados en la web. Cero configuración, solo instala las dependencias y listo.

Generación de Plantillas

Genera archivos estaticos, te permite crear estructura de trabajo para páginas estaticas, temas o plugins de WordPress o generar la estructura de un proyecto de Electron o React JS.

El desarrollo web hecho fácil

Create App CLI toma la parte complicada de la configuración y la gestión de dependencias para que te enfoques en tu aplicación.

Genera tus plantilla base

Ideal para principiantes les permite integrar la estructura desde una simple pagina hasta una aplicacion web. Obteniendo todo el control desde el principio, incluye un servidor estatico para la visualizacion de tu contenido sin tener que descargar un paquete o servidor web.

Aplicaciones de escritorio. No hay problema

Te cubrimos con eso, trabajamos con una versión unversal de ElectronJS que funciona que cualquier dispositivo en Windows y Linux de 32 o 64 bits. A partir de la version NodeJS 8, disponible con Electron Forge. Agrega las dependencias que necesites mientras construyes tu proyecto.

ReactJS, sencillo y simplificado

Sabemos lo complejo que es configurar React con JSX o TSX para versiones especificas de NodeJS, para los principiantes ofrecemos una alternativa que puede trabajar con las versiones de Webpack 4 o 5, si lo necesitas ajusta las dependencias con npm y listo.

WordPress, desde código

Te gusta programar las funcionalidades de tus temas, widgets y plugins desde el código PHP. No hay problema, existe un comando para generar la plantilla base de tu tema o plugin. Modifica como quieras y agregalo a tu servidor.
